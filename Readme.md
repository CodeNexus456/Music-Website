# 🎵 Beautify Clone

A modern and responsive Beautify** Clone** built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Beautify's web player with a clean user interface, interactive music controls, and responsive design.

---

## 🚀 Features

* 🎧 Modern Beautify-inspired UI
* 📱 Fully Responsive Design
* ▶️ Play & Pause Music
* ⏭️ Next & Previous Song Controls
* 🎵 Dynamic Song List
* 📊 Music Progress Bar
* 🔊 Volume Control
* 💚 Beautiful Hover Effects & Animations
* ⚡ Smooth User Experience

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
